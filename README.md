generator-h is a simple React generator with Webpack 2 and ES2015.

## Features

- React
- Webpack 2
- ES2015

## Requirements

- Node.js v4+ and npm

## Getting Started

```
npm install -g yo
npm install -g generator-h
mkdir <project-name> && cd <project-name>
yo h
```

## Commands

`yo h` 

> Shows a wizard for generating a new React project
